## Live Demo

- **Portfolio:** https://avneesh-tech.vercel.app

## 🚀 Features

- **Live Terminal Simulator:** Fully operational simulated `/bash` terminal with active command execution (`projects`, `ping`, `whoami`, `techstack`, `clear`, `quit`).
- **Data-Stream Marquee Engine:** Custom infinite-scrolling cyber background representing live server execution logs (`NODE.JS`, `MONGODB`, `REDIS`, `EXPRESS`, `AWS`).
- **Interactive Copyable Résumé:** A meticulously stylized JavaScript object `avneesh.ts` that explicitly details my skills, roles, and focus areas, equipped with a 1-click clipboard function.
- **Cyber-Cyan Aesthetic Overhaul:** A cohesive tech-heavy UI shifting entirely into a deep slate black and raw cyber cyan (`#00d8ff`) palette, dropping standard saturated templates for an authentic OLED hacking experience.
- **GSAP Magnetic Physics:** Heavy usage of physics-based cursor interactions, magnetic anchor elements, and fluid `ScrollSmoother` page navigation.
- **Dynamic Mouse Spotlight:** Interactive radial gradients that calculate cursor geometry tracking to build a dynamic flashlight over a faint architectural CSS network grid.

## 🛠️ Built With

- **Frameworks/Libraries:** React.js, React Native, Vite.js
- **Styling:** Custom Vanilla CSS Grid / Flexbox & CSS Variables
- **Animations:** GSAP (ScrollTrigger, ScrollSmoother, SplitText)
- **3D & Physics:** React Three Fiber, Rapier Physics Engine
- **Icons:** React Icons

## ⚙️ Running Locally

Want to inspect the codebase or run the simulated backend environment on your machine?

```bash
# Clone the repository
git clone https://github.com/YourUsername/avneesh-portfolio-main.git

# Navigate into the project
cd avneesh-portfolio-main

# Install dependencies
npm install

# Start the dev server
npm run dev
```

## 🔒 Security

All code strictly enforces front-end security paradigms. Environment variables and backend-specific secret endpoints are isolated strictly on the server-side, with only simulation configurations running statically here.
